Code Example: Semantic HTML

<!-- Semantic HTML5 Structure -->
<article>
  <header>
    <h1>Our Coffee Story</h1>
  </header>
  <nav aria-label="Story navigation">
    <ul>
      <li><a href="#origins">Origins</a></li>
    </ul>
  </nav>
  <section id="origins">
    <h2>Our Origins</h2>
    <p>...content...</p>
    <figure>
      <img src="..." alt="...">
      <figcaption>...</figcaption>
    </figure>
  </section>
  <aside>
    <h4>Did You Know?</h4>
    <p>...</p>
  </aside>
  <footer>
    <p>...</p>
  </footer>
</article>
HTML Features Demonstrated
  • Semantic Tags: <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>, <figure>, <figcaption>
  • Accessibility: aria-label, proper heading hierarchy (h1-h3), alt text for images
  • Lists: Ordered lists <ol>, unordered lists <ul>
  • Links: Internal navigation with <a> tags
  • Forms: Used in contact section on main page with validation
  • Tables: Used in menu section (if needed)
  • HTML5 Doctype: <!DOCTYPE html>
